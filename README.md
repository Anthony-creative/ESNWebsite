# ESN Landing Page (Vanilla HTML/CSS/JS)

Esta es una landing page moderna para nutrición deportiva construida únicamente con **HTML5, CSS3 y JavaScript puro**. Utiliza Tailwind CSS a través de CDN para los estilos y Lucide para los iconos.

## 🚀 Cómo publicar gratis en GitHub Pages

Al ser un proyecto estático (sin React ni Node.js), publicarlo es extremadamente fácil:

1. Crea un nuevo repositorio en tu cuenta de [GitHub](https://github.com/new).
2. Sube los archivos `index.html`, `style.css` y `script.js` a tu repositorio.
3. En tu repositorio de GitHub, ve a la pestaña **Settings** (Configuración).
4. En el menú de la izquierda, haz clic en **Pages**.
5. En la sección "Build and deployment", bajo "Source", selecciona **Deploy from a branch**.
6. Bajo "Branch", selecciona **main** (o master) y la carpeta **/(root)**. Haz clic en **Save**.
7. ¡Listo! En un par de minutos, GitHub te dará un enlace público (ej. `https://tu-usuario.github.io/tu-repo/`) donde cualquiera podrá ver la página.

## 💻 Cómo editar localmente

Simplemente haz doble clic en el archivo `index.html` para abrirlo en cualquier navegador web (Chrome, Safari, Edge). No necesitas instalar nada.
